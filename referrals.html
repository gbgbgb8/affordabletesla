Add Your Referral Link

If you'd like to add your Tesla referral link to the list, please follow the steps below:

  1. Visit the Affordable Tesla GitHub repository.
  2. Click on the "Issues" tab.
  3. Click on the "New issue" button.
  4. Provide a title for the issue, such as "New Referral Link".
  5. In the description, paste your referral link.
  6. Click "Submit new issue".

Once your issue has been reviewed, we'll add your referral link to our list! Thank you for contributing!

How Referral Links Work

Random Referral Link Selection:

  1. Load Referral Links:

    When the website is loaded, it fetches the list of referral links from the referrals.json file. This JSON file contains an array of different referral links that users have contributed over time.

  2. Random Selection:

    A random number generator is used to select an index from the array of referral links. This ensures that every time the page is loaded, a different referral link is chosen with equal probability.

  3. Set the Link:

    The randomly selected referral link is then set as the href attribute of the "Shop Tesla.com" button. This means when users click on this button, they will be redirected to Tesla's website using the randomly chosen referral link.

  4. Equal Opportunity:

    Since the selection process is entirely random and is based on the number of links in the referrals.json, every referral link has an equal chance of being selected each time the page is loaded. This ensures fairness and gives everyone an equal opportunity to have their referral link used by potential buyers.

  5. Update and Expansion:

    As more referral links are added to the referrals.json (for instance, through GitHub Issues as described earlier), the pool of available links grows. This ensures that the system remains dynamic and can easily accommodate more contributors over time.
